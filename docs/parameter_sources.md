# 参数来源表

## 说明

本文用于明确当前评分与模型中的参数来源类型。这里区分四类来源：

- `机制/文献方向支撑`：方向具有明确理论或文献基础。
- `局地尺度化`：方向确定后，结合新加坡海峡样本分布落地具体切点。
- `工程启发式`：用于形成稳定、可解释的平台结构，但尚未完成外部标定。
- `平台内部校准`：在当前样本与验证集上，为了让输出更稳定而做的校准。

## 评分参数

| 项目 | 当前做法 | 来源类型 | 说明 |
| --- | --- | --- | --- |
| SST 分段 | `<18` 低值，`18–25` 上升，`25–31` 平台，`31–33` 软回落，`>33` 继续回落 | 机制/文献方向支撑 + 局地尺度化 | 热带港区不适合做对称钟形，需保留暖水高平台。 |
| Salinity 分段 | `<20` 强惩罚，`20–28` 上升，`28–35` 平台，`35–38` 轻回落，`>38` 下降 | 机制/文献方向支撑 + 局地尺度化 | 盐度被视为闸门项，而不是主导增益项。 |
| Chlorophyll-a 分段 | `<0.5` 低压力，`0.5–2` 中等，`2–8` 高压力，`>8` 封顶 | 机制/文献方向支撑 + 工程启发式 | 用作生产力/传播压力代理，不做钟形最适曲线。 |
| Current 处理 | 先合成 `current_speed`，再按低流速利附着、高流速不利附着建模 | 机制/文献方向支撑 + 局地尺度化 | 不再分别给 `u / v` 打分，切点使用研究区百分位。 |
| EnvModifier 权重 | `0.40T + 0.20S + 0.25P + 0.15H` | 工程启发式 | 权重体现“温度主导、盐度闸门、叶绿素代理、海流约束”的结构逻辑。 |
| EnvAdj | `0.85 + 0.30 × EnvModifier` | 工程启发式 | 以 `1` 为中心，允许环境既增强也削弱行为风险。 |
| MaintenanceAdj | `0.90 + 0.20 × MaintenanceScore` | 工程启发式 | 维护轻修正回到 FPI 中，防止维护历史完全消失。 |
| FPI 结构 | `BehaviorExposure × EnvAdj × MaintenanceAdj` | 机制/文献方向支撑 + 工程启发式 | 体现“行为主导、环境修正、维护轻修正”。 |
| CarbonPenaltyModifier | `1 + 0.18 × MaintenanceBurden + 0.12 × PersistentExposure` | 工程启发式 | 表达潜在代价放大，不对应正式碳核算公式。 |
| ECP 结构 | `FPI × CarbonPenaltyModifier` | 工程启发式 | ECP 是代价放大器，不是第二套独立风险分。 |
| RRI 权重 | `0.40 Env + 0.25 Traffic + 0.20 StayProb + 0.15 PortAnchorageIntensity` | 工程启发式 | 用于形成可解释的区域综合风险图层。 |
| FPI/ECP 阈值 | `0.25/0.30` 与 `0.08/0.10` | 平台内部校准 | 基于新评分尺度重标，避免科学升级后大面积误判为低风险。 |

其中，ECP 的正式结构表达为：

`ECP = FPI × CarbonPenaltyModifier`

## 深度模块参数

| 模块 | 当前做法 | 来源类型 | 说明 |
| --- | --- | --- | --- |
| LSTM 输入窗口 | 最近 `8` 个 `6` 小时窗口预测下一个窗口 | 工程启发式 | 结合当前 15 天样本长度与可用训练样本量确定。 |
| LSTM 风险阈值 | 验证集校准得到的风险标签阈值 | 平台内部校准 | 只适用于当前平台预测口径。 |
| LSTM 可预测对象 | 需具备足够连续历史窗口的对象 | 方法边界 | 并非所有船都能预测。 |
| Autoencoder 结构 | 全连接自编码器 + 重建误差筛查 | 方法支撑 | 回答的是“是否偏离常规模式”，不是“是否已发生真实污损”。 |
| 异常分级阈值 | `0.35` 高度异常，`0.12` 需复核，加观测不足门槛 | 工程启发式 + 平台内部校准 | 用于总览筛查与单船解释，不应解释为外部行业标准。 |
