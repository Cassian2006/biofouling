# 指标设计草案

## FPI

FPI（Fouling Propensity Index）用于表达某船在给定时间窗内累积生物污损的倾向，首版建议由以下因子组合构成：

- 停泊时长
- 锚泊时长
- 低速航行时长
- 高温海域暴露
- 高叶绿素海域暴露
- 港口访问频率

## ECP

ECP（Emission Cost Penalty）用于表达潜在污损造成的相对能耗与碳排惩罚，第一阶段以相对惩罚值为主，不输出工业级绝对结果。

## RRI

RRI（Regional Risk Index）用于表达研究区空间单元的污损促进风险，首版建议综合：

- 环境适宜性
- 交通密度
- 低速或停泊概率
- 港口与锚地暴露强度

## 原则

- 优先可解释
- 优先可视化
- 优先可复现
- 优先相对排序能力
