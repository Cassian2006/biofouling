# 科学性 Review

## 目的

本文档用于明确当前平台各个评分与模型模块的科学边界，回答三件事：

1. 哪些部分具有明确的文献或机理支撑
2. 哪些部分属于工程启发式设计
3. 哪些输出只适合相对排序，不适合绝对判断

## 总体结论

当前平台更适合被定义为：

- `可解释的相对风险排序平台`
- `机制约束 + 工程实现` 的竞赛级分析系统

而不是：

- 直接估计真实船底污损厚度的生态模型
- 直接输出工业级绝对燃油惩罚或绝对碳排代价的运营系统

## 模块逐项审查

### 1. FPI

#### 有机理支撑的部分

- 行为暴露是主因，环境因子是修正项，这与 [IMO Biofouling](https://www.imo.org/en/OurWork/Environment/Pages/Biofouling.aspx) 的风险逻辑一致。
- 温度、盐度、初级生产力和局地流动条件都会影响附着生物的存活、繁殖和早期附着窗口。
- 局地流速对藤壶 cyprid 临时附着概率的影响具有明确方向支撑，可参考 [Journal of Experimental Biology: Temporary adhesion of barnacle cyprids](https://journals.biologists.com/jeb/article/211/13/2178/18261/Temporary-adhesion-of-barnacle-cyprids-effects-of/18261)。

#### 工程启发式设计

- `sst / salinity / chlorophyll-a / current_speed` 的具体分段切点不是文献直接给出的统一真阈值，而是“文献定方向，研究区定尺度”的工程落地。
- `EnvModifier = 0.40T + 0.20S + 0.25P + 0.15H` 是工程分配权重，不是文献原文直接给出的固定系数。
- `FPI = BehaviorExposure × EnvAdj × MaintenanceAdj` 的结构是机制优先的工程实现，用于避免环境单独抬分造成的假阳性。

#### 适用边界

- 适合：当前窗口内的相对风险排序、单船暴露诊断、同批对象比较
- 不适合：绝对污损厚度判断、跨海区直接比较、物种级生态结论

### 2. ECP

#### 有机理支撑的部分

- 船体污损会增加阻力并抬高燃油与排放代价，这个方向具有明确工程与海运运维常识支持。
- 将 ECP 作为 FPI 之后的代价放大器，而不是与 FPI 并行的另一套风险主分数，结构上更合理。

#### 工程启发式设计

- 当前 `CarbonPenaltyModifier` 仍是代理结构，不是工业级阻力或燃油模型。
- 其中维护负担与持续暴露的系数是工程参数，用于表达“污损倾向一旦形成，潜在代价会被进一步放大”。

#### 适用边界

- 适合：维护优先级判断、辅助展示潜在能耗代价
- 不适合：真实燃油结算、真实碳核算、正式航运运营成本预测

### 3. RRI

#### 有机理支撑的部分

- 区域热点确实应由环境条件、交通密度、停留概率和港口 / 锚地暴露共同表达。
- RRI 更接近“区域促进条件”而不是“真实污染结果”，这一定位是科学上稳妥的。

#### 工程启发式设计

- 当前网格尺度、组合权重与风险分级是平台级表达设计。
- `StayProb` 和 `PortAnchorageIntensity` 的拆分是为避免重复计分，是解释驱动的工程优化。

#### 适用边界

- 适合：热点识别、区域比较、优先巡查
- 不适合：绝对生态强度判断

### 4. 异常检测

#### 有方法支撑的部分

- 使用无监督异常检测去发现“明显偏离常规暴露模式”的对象，方法上成立。
- 当前 Autoencoder 的含义是学习常规模式，再用重建误差识别偏离对象。

#### 工程启发式设计

- 当前异常类型识别仍然是 `Autoencoder 分数 + 规则归类`，不是单独训练的监督式分类器。
- `highly_abnormal / suspicious / normal / observation_insufficient` 的分级规则属于工程分层。

#### 适用边界

- 适合：筛查异常对象、总览页预警、单船解释增强
- 不适合：直接认定“已经发生严重污损”

### 5. 时序预测

#### 有方法支撑的部分

- 使用历史 AIS 行为与环境暴露序列预测下一时间窗风险，属于合理的短期时序建模方向。
- LSTM 作为轻量时序模型，适合当前竞赛级场景。

#### 工程启发式设计

- 当前预测目标仍是 `FPI proxy`，不是外部观测到的真实污损值。
- 风险标签阈值来自验证集校准，因此是平台内标签，不是外部标准。

#### 适用边界

- 适合：短期趋势预测、风险预警
- 不适合：长期生态演化预测、绝对生物量预测

## 当前最重要的答辩口径

建议统一使用以下表述：

- `FPI / ECP / RRI` 是可解释代理指标，不是绝对真值
- 环境因子的作用方向由机理和文献支撑，具体切点和权重是热带港区场景下的工程先验
- 异常检测和时序预测增强的是筛查与预警能力，不替代规则链路本身

## 当前仍待加强的部分

- 更细粒度的文献化参数校准
- 不同时间窗与不同海区的迁移验证
- 更真实的外部校验样本
- 对 ECP 的燃油与碳代价映射校准
