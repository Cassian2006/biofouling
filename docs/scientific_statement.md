# 科学性说明

## 文档定位

本文是面向竞赛汇报与答辩的正式材料，用于统一回答以下问题：

1. 为什么 FPI 采用“行为主导、环境修正、维护轻修正”的结构。
2. 为什么 ECP 只能被解释为相对惩罚，而不是正式碳核算。
3. 为什么 RRI 是区域综合风险图层，而不是“真实污损地图”。
4. LSTM 与异常检测分别回答什么问题，哪些对象能预测，哪些对象只是异常筛查。
5. 当前哪些部分已经有真实机制支撑，哪些部分仍是启发式近似，以及这些工作我们做到了哪一步。

## 1. FPI 的科学口径

### 为什么是行为主导

船体污损积累首先取决于暴露是否真的发生，也就是船是否长期低速、停留、锚泊、靠港。环境可以改变暴露后果，但如果船只始终高速通过，环境再适宜也不应被直接打成高污损倾向。

因此，FPI 的主体仍然是 `BehaviorExposure`。

### 为什么环境只做修正

环境项回答的是“这片水体在当前条件下，会不会放大或削弱行为暴露的后果”。因此，环境不应该单独飞出一套与行为平行的主分，而应该作为对行为暴露的增强或削弱。

当前结构为：

```text
EnvModifier = 0.40T + 0.20S + 0.25P + 0.15H
EnvAdj = 0.85 + 0.30 × EnvModifier
```

这意味着环境既可以削弱风险，也可以增强风险，但不会替代行为本身。

### 为什么维护是轻修正

维护历史会改变同样暴露下的污损累积速度。刚维护过的船与长期未维护的船，不应在相同行为和环境下得到完全相同的污损倾向。

但维护也不应成为主导项，因此当前做法是将其保留为轻修正：

```text
MaintenanceAdj = 0.90 + 0.20 × MaintenanceScore
FPI = BehaviorExposure × EnvAdj × MaintenanceAdj
```

### 结论

FPI 当前回答的是：

- 在当前时间窗内，这艘船在其实际行为模式、所处环境和维护背景下，是否更容易形成污损积累倾向。

FPI 当前不回答：

- 真实污损厚度是多少。
- 船底已经长了多少。
- 不同海区间无校准的绝对风险比较。

## 2. ECP 的科学口径

### 为什么不能做正式碳核算

当前 ECP 并没有接入船型阻力曲线、真实主机工况、真实油耗模型和正式碳排放核算规则。因此它不具备被解释为正式碳核算结果的条件。

### 它当前真正表达什么

ECP 的结构是：

```text
CarbonPenaltyModifier = 1 + 0.18 × MaintenanceBurden + 0.12 × PersistentExposure
ECP = FPI × CarbonPenaltyModifier
```

它表达的是：

- 当一艘船已经具有较高污损倾向时，长期未维护和持续高暴露会进一步放大潜在能耗与运营代价。

### 结论

ECP 当前适合被解释为：

- 相对能耗惩罚指标
- 维护优先级的辅助排序项

ECP 当前不适合被解释为：

- 正式碳核算
- 真实燃油消耗
- 正式排放申报结果

## 3. RRI 的科学口径

### 为什么不是“真实污损地图”

区域层没有真实船体污损观测网格，因此当前不可能直接画出“真实污损地图”。当前区域图层只能表达：

- 这片区域是否更容易形成高暴露、高停留、高传播压力和高参考点聚集的综合风险条件。

### 它当前真正表达什么

RRI 结构为：

```text
RRI = 0.40 × EnvModifier + 0.25 × Traffic + 0.20 × StayProb + 0.15 × PortAnchorageIntensity
```

对应的是：

- 环境适宜性
- 交通疏密
- 停留概率
- 港口/锚地强度

### 结论

RRI 当前适合被解释为：

- 区域综合风险图层
- 巡查和研判的热点参考层

RRI 当前不适合被解释为：

- 真实污损现状图
- 区域生态实测图

## 4. LSTM 与异常检测分别回答什么

### LSTM 预测模块

LSTM 回答的问题是：

- 这艘船在最近一段连续历史暴露之后，下一时间窗的 FPI proxy 会往哪里走。

它的本质是：

- 短期趋势预测
- 下一窗口预警

它不是：

- 长期生态预测
- 真实污损厚度预测

### 哪些对象能预测

只有具备足够连续历史窗口的对象，才能进入 LSTM 预测。当前平台要求最近有足够连续的 `6` 小时窗口序列，因此并不是所有船都能预测。

如果一艘船历史不足，平台会明确提示：

- 当前对象缺少足够连续历史窗口，暂未生成 LSTM 预测。

### 异常检测模块

异常检测回答的问题是：

- 这艘船的暴露模式是否明显偏离当前样本中的常规模式。

它的本质是：

- 异常筛查
- 总览预警
- 单船解释增强

它不是：

- 真实污损已发生的证明
- 真实故障诊断

## 5. 哪些有真实机制支撑，哪些仍是启发式近似

### 当前已有真实机制支撑的部分

- 行为暴露是污损倾向主因。
- 海温、盐度、生产力代理和水动力条件会影响附着窗口。
- 流速影响附着应按 `current_speed` 而不是 `u / v` 单独理解。
- 污损倾向升高后，潜在能耗和维护代价会被放大。
- 短期时序模型适合做下一窗口风险预测。
- 无监督异常检测适合做“偏离常规模式”的筛查。

### 当前仍属启发式近似的部分

- 各环境变量的具体分段切点。
- `EnvModifier` 与 `RRI` 的组合权重。
- `CarbonPenaltyModifier` 的具体系数。
- FPI、ECP 的分级阈值。
- 异常分级阈值与异常类型规则。
- LSTM 风险标签阈值与 Autoencoder 分层阈值。

## 7. 推荐答辩表述

- FPI、ECP、RRI 是可解释代理指标，不是绝对真值。
- FPI 体现的是行为主导、环境修正、维护轻修正。
- ECP 体现的是潜在代价放大，不是正式碳核算。
- RRI 体现的是区域综合风险条件，不是实测污损地图。
- LSTM 用于短期趋势预测，异常检测用于异常筛查，两者不替代规则主链路。
