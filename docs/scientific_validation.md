# 科学验证补充

## 目的

这一轮补充不再增加新模型，而是回答一个更适合答辩的问题：

- 当前 `science-v2` 评分在小幅参数变化下是否稳定。
- 四个环境机制分量里，哪一项对排序结果影响更大。
- 现有结论更像“稳健排序”，还是“高度依赖某几个启发式系数”。

## 方法

本验证基于已经冻结的 `20260115 to 20260130` 窗口，但不会覆盖竞赛主版本展示结果。

### 1. 敏感性分析

在保持数据不变的前提下，只对评分结构做小幅调整：

- 温度权重上调
- 叶绿素权重上调
- 环境修正收窄
- 环境修正放宽
- 维护修正减弱
- 维护修正加强

观测指标包括：

- Top-20 船舶排序重合度
- Top-50 船舶排序重合度
- 推荐类别变化率
- 三类建议数量变化

### 2. 分量消融分析

分别移除四个环境机制分量中的一个，再观察：

- 平均 FPI 变化量
- Top-20 船舶排序重合度
- 推荐类别变化率

这里的“移除”不是删除整套模型，而是把对应分量从 `EnvModifier` 的权重组合中拿掉，再对剩余分量归一化。

## 当前判断口径

如果某个小幅改动就导致：

- Top-20 大面积洗牌
- 推荐类别显著翻转
- 大量对象从“需关注”掉到“低即时关注”

那就说明当前模型仍然过度依赖启发式常数。

相反，如果结果表现为：

- Top-20 基本稳定
- 只有阈值附近对象发生有限翻转
- 去除某一分量后变化可解释

那就可以更有底气地说：当前模型虽然仍是工程代理指标，但已经具备一定稳健性。

## 当前结果（20260115 to 20260130）

### 基线分布

- `Prioritize cleaning assessment = 187`
- `Monitor exposure trend = 207`
- `Low immediate concern = 243`

### 敏感性分析结论

在当前 `science-v2` 结构下，六组小幅参数扰动都保持了：

- `Top-20 overlap = 1.00`
- `Top-50 overlap = 1.00`

这说明当前高优先级对象的排序，对温度权重、叶绿素权重、环境修正幅度和维护修正幅度的小幅调整都比较稳定。

推荐类别变化率最高的是：

- `环境修正放宽 = 0.78%`

也就是说，小幅调权会影响阈值附近的少量对象，但没有导致大面积洗牌。

### 分量消融结论

去掉单个分量后，排序仍然保持稳定，但平均分数和建议分布会出现可解释变化：

- 去掉温度分量：`mean_fpi_shift = -0.0104`
- 去掉盐度分量：`mean_fpi_shift = -0.0040`
- 去掉叶绿素分量：`mean_fpi_shift = +0.0097`
- 去掉水动力分量：`mean_fpi_shift = +0.0028`

当前最值得注意的两点是：

1. 温度仍然是最强的抑制/增强方向之一，拿掉后高风险对象会明显回落。
2. 叶绿素当前更像“近岸高生产力压力代理”，拿掉它会让一部分对象的环境修正下降约束减弱，因此平均 `FPI` 反而略有上升。

这说明当前模型不是“每个环境因子都在机械抬分”，而是已经形成了彼此制衡的组合结构。

## 当前阶段定位

这部分属于“学术增强”和“答辩增强”，不是竞赛主页面的默认展示内容。

它的作用是帮助回答：

- 为什么这套评分不是拍脑袋。
- 哪些部分已经比较稳。
- 哪些部分仍然是先验假设，需要未来继续校准。
