# 港口与锚地参考层计划

这条线服务于 1.0 的“强体验、强可视化展示”目标。

## 目标

让平台不只回答：

- 风险热点在哪里
- 船舶轨迹在哪里

还开始回答：

- 这些热点或轨迹靠近哪个港口或锚地
- 当前高风险更像是近港暴露、锚地等待，还是普通过境活动

## 当前接入策略

当前先用轻量参考层，不等完整港口数据库。

当前种子文件：

- [singapore_reference_sites.csv](/c:/Users/cai%20yuan%20qi/Desktop/biofouling/data/reference/singapore_reference_sites.csv)

当前包含两类点位：

- `port`
- `anchorage`

当前用途：

- 给区域热点匹配最近参考点
- 给单船详情匹配最近港口 / 锚地
- 给前端页面增加“地理语义解释”，避免只剩经纬度

## 后续扩展

后续可以继续补：

- 更细的港区范围
- 航道中心线
- 到港 / 离港事件
- 锚地分区与服务类型

## 1.0 阶段的合理边界

1. 先有轻量可解释参考层
2. 先让区域页和单船页能展示最近港口 / 锚地
3. 后续再逐步升级为更正式的参考地理图层
